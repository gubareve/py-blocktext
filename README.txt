The pyblocktext package makes ascii art super easy!

print ("A") -->

           
     /\    
    /  \   
   / /\ \  
  / ____ \ 
 /_/    \_\
           
           





Install

---------------------------

$ pip install pyblocktext 

or 

$ python -m pip install pyblocktext

Usage

---------------------------

import pyblocktext

blocktext = pyblocktext.blocktext

print blocktext("a")

ONLY WORKS FOR INDIVIDUAL CHAR'S
